This section describes how to edit project annotations.

Project annotations are key-value pairs that add extra metadata or descriptive information to a project.

== Steps

include::../../../../_custom/workspaceManagement/workspaceManagement-oper-openWorkspacePage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]


. On the right side of the project you want to operate, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], and then select **Edit Annotations** from the drop-down list.

. In the **Edit Annotations** dialog box, set the annotation key-value pairs, and then click **OK**.
+
--
* Click **Add** to set multiple annotations.

* Click image:/images/ks-qkcp/zh/icons/trash-light.svg[trash-light,18,18] on the right side of a created annotation to delete it.
