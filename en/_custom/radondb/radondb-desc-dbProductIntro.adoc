This section introduces how to manage the RadonDB {ks_db_product} database application.

RadonDB {ks_db_product} is an enterprise-level database product based on {ks_db_product}, now available in the KubeSphere App Store. You can install the RadonDB {ks_db_product} application in the KubeSphere web console, thereby deploying the {ks_db_product} database in a containerized manner and managing the {ks_db_product} database through a graphical interface.
