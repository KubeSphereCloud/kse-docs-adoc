:ks_permission: **Role Management**
:ks_navigation: **Workspace Settings > Workspace Roles**


This section describes how to edit workspace role permissions.

You can only edit the permissions of user-created workspace roles. The permissions of built-in workspace roles cannot be edited.

. In the **Edit Permissions** dialog box, set the permissions for the workspace role, then click **OK**.
