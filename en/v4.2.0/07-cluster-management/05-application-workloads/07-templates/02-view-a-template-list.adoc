* The application workload template list provides the following information:
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name and description of the application workload template.

|Type
|The type of the application workload template.

|Project
|The project to which the application workload template belongs.

|Creation Time
|The creation time of the application workload template.
|===
--

* Select a project from the drop-down list in the upper-left corner of the list to view application workload templates in a specific project.

* Click the search box above the list and set search criteria to search for application workload templates by name and type.
