Resource Management provides Kubernetes resource management from a tenant's perspective under a physical cluster, previews various resources through a directory tree format, supports viewing and downloading resource YAML files, and synchronizes them to a code repository.

// |Note

// |This feature is only available in KubeSphere ``Standard Edition, Advanced Edition, and Enterprise Edition``.

// |===
