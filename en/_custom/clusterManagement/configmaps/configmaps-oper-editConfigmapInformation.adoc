. On the right side of the ConfigMap you want to operate, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Edit Information** from the dropdown list.

. In the **Edit Information** dialog box, set the alias and description for the ConfigMap, then click **OK**.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Alias
|The alias of the ConfigMap. Aliases for different ConfigMaps can be the same.
// The alias can only contain Chinese characters, letters, numbers, and hyphens (-). It must not start or end with a hyphen (-) and can be up to 63 characters long.

|Description
|The description of the ConfigMap. The description can contain any characters, up to a maximum of 256 characters.
|===
