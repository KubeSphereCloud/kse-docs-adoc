This section describes how to delete a project role.

You can delete project roles created by users. Built-in project roles cannot be deleted.

== Prerequisites

:relfileprefix: ../../../../
* {empty}
include::../../../../_custom/projectManagement/projectManagement-prer-requiredPermission_v42.adoc[]

:relfileprefix: ./

* If the project role you want to delete has already been assigned to project members, you need to remove those project members or modify their roles.
