本节介绍如何添加对集群和企业空间的快捷访问，以便快速进入对应资源的管理页面。

== 前提条件

已获取 KubeSphere Web 控制台的用户名和密码。

== 操作步骤

. 登录 KubeSphere Web 控制台。
. 在集群列表和企业空间列表页面，点击image:/images/ks-qkcp/zh/icons/star_none.svg[star_none,18,18]，即可添加对应集群和企业空间到**快捷访问栏**和**快捷访问**区域。点击image:/images/ks-qkcp/zh/icons/star.svg[star,18,18]，即可从快捷访问中移除集群和企业空间。
+
也可在**快捷访问**区域点击image:/images/ks-qkcp/zh/icons/star.svg[star,18,18]，移除已添加的集群和企业空间。
