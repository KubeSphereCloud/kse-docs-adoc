This section describes how to view and add licenses for the KubeSphere platform and extensions.

**KubeSphere License**: Used to activate the KubeSphere platform. All versions of KubeSphere require activation via a license. Please visit the KubeSphere official website link:https://kubesphere.com.cn/kse/[to purchase a license], or link:https://kubesphere.com.cn/apply-license[apply for a free Community Edition license].
// Starting from KubeSphere v4.2.0, KubeSphere is divided into four editions: Community Edition, Standard Edition, Advanced Edition, and Enterprise Edition.

**Extension License**: Used to activate certain paid extensions. Only the authorization information for extensions is displayed here.
// For more information about extension licensing, please visit the extension details page in the Extensions Center and contact the extension provider for support.
