. On the right side of the Ingress you want to operate, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Edit Info** from the dropdown list.

. In the **Edit Info** dialog box, set the alias and description for the Ingress, then click **OK**.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Alias
|The alias for the Ingress. Different Ingresses can have the same alias.
// The alias can only contain Chinese characters, letters, numbers, and hyphens (-). It must not start or end with a hyphen (-) and can be up to 63 characters long.

|Description
|The description for the Ingress. The description can contain any characters and is limited to 256 characters.
|===
