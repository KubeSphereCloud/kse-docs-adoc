:ks_navigation: **配置 > 保密字典**

本节介绍如何进行密码过期设置，以便强制定期修改密码，在密码过期前提醒。


== 前提条件

您需要加入一个集群并在集群中具有 **cluster-admin** 角色。

== 操作步骤

. 以具有 **cluster-admin** 角色的用户登录 KubeSphere Web 控制台并进入您的集群。

+

include::../../../../_ks_components/oper-navigate.adoc[]

. 搜索 `io.kubesphere.config.platformconfig.security`。

. 在保密字典右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**编辑设置**。

. 在**编辑设置**对话框，点击image:/images/ks-qkcp/zh/icons/pen-light.svg[pen-light,18,18]，修改各字段的值。然后点击**确定**。
+
--
[source,yaml]
----
passwordPolicy:
  expireEnabled: false
  maxAgeDays: 90
  reminderDays: 7
----

[%header,cols="1a,4a"]
|===
|参数 |描述

|`expireEnabled`
|是否启用密码过期功能，即强制定期修改密码。

|`maxAgeDays`
|如果启用密码过期功能，密码的最长有效期为 90 天。

|`reminderDays`
|如果启用密码过期功能，系统会在密码到期前 7 天开始提醒用户。

// |`forceChangeOnExpire`
// |过期时是否强制修改密码。若 `expireEnabled` 和 `forceChangeOnExpire` 均为 `true`，一旦密码过期，将强制修改密码。
|===
