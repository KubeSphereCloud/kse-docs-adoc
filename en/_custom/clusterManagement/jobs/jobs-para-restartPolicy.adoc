* Pod Settings > Restart Policy
+
--
Set the policy for the system to take when a container fails.

[%header,cols="1a,4a"]
|===
|Parameter |Description

|Recreate Pod
|The system will recreate the entire pod.

|Restart Container
|The system will restart the container.
|===
