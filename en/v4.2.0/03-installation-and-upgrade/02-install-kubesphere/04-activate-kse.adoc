After logging into the {ks_product_left} web console, you need to import a license to activate {ks_product_both} and extensions to experience all features.

[.admon.note,cols="a"]
|===
|Note

|
If you installed {ks_product_left} offline, please contact the delivery service expert to obtain the license directly.
|===

== Step 1: Purchase and Obtain the License

If you installed {ks_product_left} online, please visit the link:https://kubesphere.com.cn/[{ks_product_right} official website], click **Contact Us**, and consult via WeChat or Slack about the purchase and acquisition methods for {ks_product_both} and extension licenses.

image:/images/ks-qkcp/zh/v4.1.3/contact-us.png[, 70%]

== Step 2: Import the License

ifeval::["{file_output_type}" == "html"]
After obtaining the license, please refer to link:../../../04-platform-management/05-platform-settings/02-license/03-add-a-license/[Add a License] to import the license for {ks_product_both} and extensions.
endif::[]

ifeval::["{file_output_type}" == "pdf"]
After obtaining the license, please refer to the "License" chapter in the "{ks_product_right} Platform Management Guide" to import the license for {ks_product_both} and extensions.
endif::[]


== Step 3: Verify Activation

If the License page shows that {ks_product_both} and extensions are **Authorized**, it means the activation is successful, and you can start using {ks_product_left}.
