. On the right side of the Secret you want to operate, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Edit Information** from the drop-down list.

. In the **Edit Information** dialog box, set the alias and description for the Secret, then click **OK**.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Alias
|The alias of the Secret. Aliases for different Secrets can be the same.
// The alias can only contain Chinese characters, letters, numbers, and hyphens (-). It must not start or end with a hyphen (-) and can be up to 63 characters long.

|Description
|The description of the Secret. The description can contain any characters, up to a maximum of 256 characters.
|===
