This section describes how to roll back microservice configuration.

Each time you edit a microservice configuration, a modification record is generated. You can use these records to roll back the microservice configuration to a historical version.


== Prerequisites

* The **KubeSphere Spring Cloud** extension must be installed and enabled on the KubeSphere platform.
