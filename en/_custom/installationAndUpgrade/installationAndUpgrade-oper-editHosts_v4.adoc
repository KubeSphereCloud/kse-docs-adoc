. Log in to all cluster nodes and execute the following command to edit the **/etc/hosts** file:
+
--
// Bash
include::../../../_ks_components/code/bash.adoc[]

vi /etc/hosts

include::../../../_ks_components/code/codeEnd.adoc[]

In the **/etc/hosts** file, add the following information to configure the domain name resolution rules for the cluster nodes for the private image service (replace `<registry IP address>` with the actual IP address of the private image service, and replace the default address **dockerhub.kubekey.local** with your actual image repository address), then save the file:

// Bash
include::../../../_ks_components/code/bash.adoc[]

<registry IP address> dockerhub.kubekey.local

include::../../../_ks_components/code/codeEnd.adoc[]
