:ks_permission: **Storage Class Management**
:ks_navigation: **Storage > Storage Classes**


This section describes how to set the default storage class. After setting a default storage class, the system will use it to create PersistentVolumeClaims if no specific storage class is specified. Only one default storage class is allowed per KubeSphere cluster.
