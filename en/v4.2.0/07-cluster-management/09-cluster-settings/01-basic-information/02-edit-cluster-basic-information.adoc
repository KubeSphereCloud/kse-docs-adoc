== Steps

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. On the **Basic Information** page, click **Edit Information** on the right.

. In the **Edit Information** dialog box, set the cluster's alias, provider, description, and label, then click **OK**.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Alias
|The alias of the cluster.

|Provider
|The provider of the cluster.

|Description
|The description of the cluster. The description can contain any characters, with a maximum length of 256 characters.

|Label
|Used to identify the purpose of the cluster. The value can be **Production**, **Development**, **Testing**, or **Demo**.
|===
