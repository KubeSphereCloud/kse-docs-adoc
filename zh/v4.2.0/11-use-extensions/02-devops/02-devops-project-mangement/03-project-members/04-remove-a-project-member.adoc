本节介绍如何移除项目成员。

// Note
include::../../../../../../_ks_components/admonitions/note.adoc[]

用户被从项目中移除后，将失去项目角色提供的权限。

include::../../../../../../_ks_components/admonitions/admonEnd.adoc[]

. 在需要移除的项目成员右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后选择**移除**。

. 在**移除成员**对话框输入项目成员的用户名，然后点击**确定**。
+
--
项目成员被移除后将不再显示在项目成员列表中。
