|===
|参数 |描述

|CPU 上限
|企业空间运行在单个集群中所有容器 CPU 上限总和的最大值。

|CPU 预留
|企业空间运行在单个集群中所有容器 CPU 预留总和的最大值。

|内存上限
|企业空间运行在单个集群中所有容器内存上限总和的最大值。

|内存预留
|企业空间运行在单个集群中所有容器内存预留总和的最大值。

|持久卷声明限制
// |持久卷声明总容量限制
|企业空间中当前已创建的持久卷声明总容量，以及可以创建的持久卷声明的容量上限。

|持久卷声明
// |持久卷声明数量限制
|企业空间中当前已创建的持久卷声明总数量，以及可以创建的持久卷声明的数量上限。

|存储类关联资源限制
// |存储类关联持久卷声明总容量限制
|存储类允许创建的持久卷声明的容量上限。

|存储类关联资源
// |存储类关联持久卷声明数量限制
|存储类允许创建的持久卷声明的数量上限。
