The operational status of each node.

* **Creating**: The node is initializing.

* **Running**: The node is operating normally.

* **Unschedulable**: The node has been manually set to unschedulable by the user. If a node is in this state, pods cannot be scheduled onto it.

* **Warning**: There are warnings on the node. You can view the warning types for the node in the **Operational Status > Health Status** area on the node details page.

* **Abnormal**: The node has encountered a fault.
