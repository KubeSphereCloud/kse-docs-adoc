After logging into the KubeSphere web console, you need to import a license to activate KubeSphere and extensions to experience all features.

== Steps

. Log in to the KubeSphere web console, click **Component Dock > Platform Settings > License** to go to the License page and obtain the cluster ID of the current cluster.
+
image:/images/ks-qkcp/zh/v4.2.0/activate-ks/get-cluster-ID.png[]

. link:https://kubesphere.com.cn/apply-license[Apply for a free license for KubeSphere Community Edition]. Fill in your personal information and the cluster ID, then submit the application.
+
--
image:/images/ks-qkcp/zh/v4.2.0/activate-ks/apply-for-license.png[,50%]

The license for KubeSphere Community Edition will then be sent to your email (the email may be intercepted; if you do not receive it for a long time, please try checking your spam folder).

[.admon.note,cols="a"]
|===
|Note

|When applying for the license, an expiration period option is provided. After the license expires, if you need to continue using it, you can apply for a free license again.

|===
--

. After receiving the email, copy the license from the email. On the License page of the KubeSphere web console, click **Add License** and paste the license.
+
image:/images/ks-qkcp/zh/v4.2.0/activate-ks/enter-license.png[, 50%]

. If the License page shows that KubeSphere and extensions are **Authorized**, it means the activation is successful, and you can start using KubeSphere.
