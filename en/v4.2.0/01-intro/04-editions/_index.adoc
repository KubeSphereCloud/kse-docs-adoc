// Starting from KubeSphere v4.2.0, KubeSphere is divided into four editions: Community Edition, Standard Edition, Advanced Edition, and Enterprise Edition. The supported product features and resource quotas vary across these editions. For more information on version differences, please refer to link:https://kubesphere.io/zh/features/[KubeSphere Official Website].

Currently, only KubeSphere Community Edition and KubeSphere Enterprise Edition are available.
