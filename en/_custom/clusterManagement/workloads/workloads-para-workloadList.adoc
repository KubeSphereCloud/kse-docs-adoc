+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name and description of the workload.

|Status
|
The number of normal pod replicas, desired pod replicas, and the running status of the workload. The workload status includes the following types:

include::workloads-para-workloadStatus.adoc[]

|Project
|The project to which the workload belongs.

|Update Time
|The update time of the workload.
