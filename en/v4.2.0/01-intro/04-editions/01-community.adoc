link:https://www.kubesphere.io/zh/[KubeSphere 社区版]支持部署在任何基础设施环境，提供在线与离线安装，支持一键升级与扩容集群，提供多租户、多集群、云原生可观测性、应用生命周期、DevOps 等功能。

使用 KubeSphere 社区版，您将免费享受持续迭代的企业级功能，轻松构建云原生基石。

== 功能清单

// 和官网的功能清单页面一致，https://kubesphere.io/zh/features/

[%header,cols="1a,4a"]
|===
|功能模块 |描述

|集群管理	|跨云、跨基础设施的多个 Kubernetes 集群的集中式可视化管理与运维	
|多租户管理	|跨集群、跨项目的资源逻辑隔离、网络隔离和细粒度的访问控制能力
|应用管理 |统一的应用生命周期管理能力，对 Helm、Operator 等类型应用提供统一的抽象层
|K8s 资源管理 |提供 Web 控制台对接 Kubernetes 原生 API 快速创建与管理 Kubernetes 资源
|扩展组件管理   |采用微内核+扩展组件的架构，按需定制、扩展平台能力
|WizTelemetry 可观测 |对应用与基础设施性能及健康状态的全面洞察
|DevOps	|提供端到端的工作流，集成主流 CI/CD 工具	
|KubeSphere 网关	|基于流量的统一入口管理、七层负载均衡与多种路由策略	
|GateKeeper |可灵活配置安全策略的准入控制器
|Metrics Server	|可扩展、高效的容器资源度量源，为 Kubernetes 内置的自动扩展管道提供服务
|NVIDIA GPU Operator	|在 Kubernetes 上创建、配置和管理 GPU
|DeepSeek	|基于 ollama 运行 DeepSeek-R1 模型，对外提供 API 服务
|Grafana	|开放且可组合的观测和数据可视化平台
|Loki	|可扩展、高可用、多租户日志聚合系统
|Higress	|基于 Istio 和 Envoy 的 API 网关
|OB-Operator	|管理 Kubernetes 集群中的 OceanBase 相关资源
|OceanBase Dashboard  |OceanBase 交互式管理应用
|Fluid |一个开源的、原生 Kubernetes 的分布式数据集编排器和加速器
|===

== 资源配额

KubeSphere 社区版最高可支持 128 vCPU 的集群规模，且仅支持 1 个集群。
