. On the PersistentVolumeClaim details page, view the resource attributes of the PersistentVolumeClaim in the **Attributes** area.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Project
|The project to which the PersistentVolumeClaim belongs.

|Status
|
include::persistentVolumeClaims-para-status.adoc[]

|Capacity
|The storage capacity requested by the PersistentVolumeClaim.

|Access Mode
|
include::persistentVolumeClaims-para-accessMode.adoc[]

|Storage Class
|The storage class requested by the PersistentVolumeClaim.

|Provisioner
|
include::../persistentVolumes/persistentVolumes-para-provisioner.adoc[]

|Persistent Volume
|The PersistentVolume bound to the PersistentVolumeClaim.

|Creation Time
|The creation time of the PersistentVolumeClaim.

|Creator
|The user who created the PersistentVolumeClaim. If this field is empty, it indicates the PersistentVolumeClaim was created automatically by the system.
|===

. On the PersistentVolumeClaim details page, view the resource usage of the corresponding PersistentVolume and the mounted pods on the **Resource Status** tab.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Persistent Volume
|Displays the used capacity percentage, remaining capacity, and total capacity of the PersistentVolume.

|Mounted Pods
|
Displays the pods that have mounted the PersistentVolume corresponding to this PersistentVolumeClaim.

. On the PersistentVolumeClaim details page, view the snapshots of the PersistentVolume corresponding to the PersistentVolumeClaim on the **Snapshots** tab.
+
--

[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name, alias, and snapshot class of the snapshot.

|Status
|The status of the snapshot.

* **Creating**: The snapshot is being created.

* **Creation Failed**: The snapshot creation failed.

* **Creation Successful**: The snapshot was created successfully.

* **Deleting**: The snapshot is being deleted.

|Capacity
|The capacity size of the snapshot.

|Creation Time
|The creation time of the snapshot.
|===
