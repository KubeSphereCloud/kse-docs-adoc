This section describes how to set up email notifications.

== Prerequisites

* You need to have the **platform-admin** role in the KubeSphere platform. For more information, refer to link:../../../../../05-users-and-roles/01-users/[Users] and link:../../../../../05-users-and-roles/02-platform-roles/[Platform Roles].

* The **WizTelemetry Notification** extension needs to be installed and enabled on the KubeSphere platform.

. On the **Notification Channels** page, click the **Email** tab, configure the email notification parameters, and then click **OK**.
+
--
* Server Settings

[%header,cols="1a,4a"]
|===
|Parameter |Description

|SMTP Server Address
|The address and port number of the SMTP server.

|Use SSL Secure Connection
|Whether to enable SSL secure connection.

|SMTP Username
|The login username for the SMTP server.

|SMTP Password
|The login password for the SMTP server.

|Sender Email Address
|The email address used to send notifications.
|===

Recipient settings for emails can be configured on the **Notification Subscriptions** page.
