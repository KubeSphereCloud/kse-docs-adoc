本节分别介绍了如何升级 KubeSphere 企业版 和 KubeSphere 开源版。

**升级 KubeSphere 企业版**：仅支持从 KubeSphere 企业版 v4.1.2 或 v4.1.3 升级至 v4.2.0，不支持从 v3.5 直接升级至 v4.2.0。若要从 v3.5 升级至 v4.2.0，请联系 KubeSphere 的交付服务专家获取升级文档，先升级至 v4.1.2。

**升级 KubeSphere 开源版**：KubeSphere 开源版可直接升级为 KubeSphere 社区版。支持从 KubeSphere 开源版 v3.x 或 v4.x 免费升级到 KubeSphere 社区版 v4.2.0。
