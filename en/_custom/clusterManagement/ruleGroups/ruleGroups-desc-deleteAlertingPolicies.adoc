This section describes how to delete an alert rule group.

// Note
include::../../../../_ks_components/admonitions/note.adoc[]

KubeSphere only supports deleting custom rule groups and does not support deleting built-in rule groups.

include::../../../../_ks_components/admonitions/admonEnd.adoc[]

// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

Rule groups cannot be recovered after deletion. Please perform this operation with caution.

include::../../../../_ks_components/admonitions/admonEnd.adoc[]
