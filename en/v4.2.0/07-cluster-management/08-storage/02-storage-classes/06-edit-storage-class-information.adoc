This section describes how to edit the alias and description of a storage class.

You can edit the alias and description of a storage class. KubeSphere does not support editing the name of a storage class.

== Steps

include::../../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

. On the right side of the storage class you want to operate on, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Edit Information** from the dropdown list.

. In the **Edit Information** dialog box, set the alias and description for the storage class, then click **OK**.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Alias
|The alias of the storage class. Different storage classes can have the same alias.

|Description
|The description of the storage class. The description can contain any characters, up to a maximum of 256 characters.
|===
