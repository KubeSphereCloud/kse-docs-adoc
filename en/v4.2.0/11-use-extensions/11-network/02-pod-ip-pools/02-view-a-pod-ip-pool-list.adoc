== Steps

+

include::../../../../../_ks_components/oper-navigate.adoc[]

+

* The Pod IP pool list provides the following information:
+
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name of the Pod IP pool.

|Status
|The status of the Pod IP pool.

|CIDR
|The CIDR block contained in the Pod IP pool.

|Used IP Addresses
|The number of used IP addresses and the total number of IP addresses in the Pod IP pool.

|IP Pool Block Size
|The CIDR size of the block used by the IP pool for IP allocation.

|Overlay Mode
|The communication mode between pods. Supports IPIP mode and VXLAN mode. If not set, BGP mode is used.

|NAT Outgoing
|Whether cross-pool communication via NAT is allowed.
|===
+
include::../../../../../_ks_components/admonitions/note.adoc[]

The Pod IP pool **default-ipv4-ippool** is automatically created after KubeSphere installation.

include::../../../../../_ks_components/admonitions/admonEnd.adoc[]
