The KubeSphere platform provides the following built-in workspace roles:

[%header,cols="1a,4a"]
|===
|Role |Description

|workspace-admin
|Workspace administrator, has full permissions within the workspace.

|workspace-self-provisioner
|Workspace self-provisioner, can create projects and view workspace settings within the workspace, and has management permissions for projects they create.

|workspace-viewer
|Workspace viewer, can view all resources within the workspace.

|workspace-regular
|Workspace regular user, only has permission to view workspace settings within the workspace. This role is typically granted to project members who do not require other workspace permissions.
|===
