// Starting from KubeSphere v4.2.0, KubeSphere is divided into four editions: Community Edition, Standard Edition, Advanced Edition, and Enterprise Edition. The product features and resource quotas supported by each edition differ. For more information on version differences, please refer to the link:https://kubesphere.io/zh/features/[KubeSphere official website].

Currently, only KubeSphere Community Edition and KubeSphere Enterprise Edition are available.
