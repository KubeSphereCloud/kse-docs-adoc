本节介绍如何上架应用模板。

[.admon.attention,cols="a"]
|===
|注意

|
在上架应用模板之前，其所包含的应用版本必须至少有一个是**已发布**状态。
|===

== 前提条件

* {empty}

* KubeSphere 平台需要安装并启用 **KubeSphere 应用商店管理**扩展组件。
