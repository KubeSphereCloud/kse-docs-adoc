|Kernel Version
|The kernel version of the Redis database. Currently supports Redis 6.2.5 and Redis 7.0.4.

|Resources
|The CPU and memory resources reserved by the system for the Redis database. This also represents the upper limit of CPU and memory resources that the Redis Sentinel database can use.

|Storage Class
|The storage class corresponding to the storage system used by the Redis database. If there is no suitable storage class in the dropdown list, you need to contact the platform administrator to create one.

|Volume
|The size of the volume used by the Redis database, in GiB.

|Redis Node Count
|The number of nodes in the Redis cluster. Valid values are **3**, **5**, **7**.

|Password
|The access password for the Redis database. The password can only contain uppercase letters, lowercase letters, numbers, and special characters (!@#$%^&*()./;), and must be between 6 and 31 characters in length.
