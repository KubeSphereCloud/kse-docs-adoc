This section describes how to delete alert rule groups.

[.admon.note,cols="a"]
|===
|Note

|
KubeSphere only supports deleting custom rule groups and does not support deleting built-in rule groups. You can reset built-in rule groups.
|===

[.admon.warning,cols="a"]
|===
|Warning

|
Rule groups cannot be recovered after deletion. Please perform this operation with caution.
|===


== Prerequisites

You need to have the **platform-admin** role in the KubeSphere platform. For more information, please refer to link:../../../../../../05-users-and-roles/01-users/[Users] and link:../../../../../../05-users-and-roles/02-platform-roles/[Platform Roles].
