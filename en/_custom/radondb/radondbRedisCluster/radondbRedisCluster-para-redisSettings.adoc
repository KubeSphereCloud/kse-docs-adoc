|Kernel Version
|The kernel version of the Redis database. Currently supports Redis 6.2.5 and Redis 7.0.4.

|Resources
|The CPU and memory resources reserved by the system for the Redis database. This is also the upper limit of CPU and memory resources that the Redis Cluster database can use.

|Storage Class
|The storage class corresponding to the storage system used by the Redis database. If there is no storage class in the drop-down list that meets your needs, you need to contact the platform administrator to create a storage class.

|Volume
|The volume size used by the Redis database, in GiB.

|Number of Shards
|The number of proxy nodes for the Redis database. The value range is from 3 to 50.

|Password
|The access password for the Redis database. The password can only contain uppercase letters, lowercase letters, numbers, and special characters (!@#$%^&*()./;). The length must be between 6 and 31 characters.
