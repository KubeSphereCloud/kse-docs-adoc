. 在**持久卷声明**页签查看使用该存储类创建的持久卷声明。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|持久卷声明的名称和别名。

|状态
|
include::../../../../_custom/clusterManagement/persistentVolumeClaims/persistentVolumeClaims-para-status.adoc[]

|容量
|持久卷声明请求的存储容量大小。

|挂载状态
|持久卷声明是否已经挂载到容器组。

|项目
|持久卷声明所属的项目。

|创建时间
|持久卷声明的创建时间。
|===
