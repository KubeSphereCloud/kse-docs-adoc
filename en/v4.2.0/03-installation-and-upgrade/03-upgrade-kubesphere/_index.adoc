This section describes how to upgrade KubeSphere Enterprise and KubeSphere Open Source respectively.

**Upgrade KubeSphere Enterprise**: Only supports upgrading from KubeSphere Enterprise v4.1.2 or v4.1.3 to v4.2.0. Upgrading directly from v3.5 to v4.2.0 is not supported. To upgrade from v3.5 to v4.2.0, please contact KubeSphere delivery service experts to obtain the upgrade documentation and upgrade to v4.1.2 first.

**Upgrade KubeSphere Open Source**: KubeSphere Open Source can be directly upgraded to KubeSphere Community Edition. Supports free upgrade from KubeSphere Open Source v3.x or v4.x to KubeSphere Community Edition v4.2.0.
