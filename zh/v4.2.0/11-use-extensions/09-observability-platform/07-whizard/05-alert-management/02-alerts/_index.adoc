本节介绍如何查看全局告警规则组生成的告警。

**全局告警**的**内置告警**和**自定义告警**页面显示系统根据全局**内置规则组**和全局**自定义规则组**生成的全局告警。当全局告警规则组中设置的指标满足预设的条件和持续时间时，系统将生成告警。当指标不再满足预设的条件时，告警将被系统自动清除。
