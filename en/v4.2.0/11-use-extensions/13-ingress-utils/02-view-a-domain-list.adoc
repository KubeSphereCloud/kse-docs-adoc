* 域名列表提供以下信息：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|域名的名称和描述。

|域名
|域名地址，支持使用通配符进行匹配。
|===
--

* 在列表上方点击搜索框并设置搜索条件，可按名称和域名搜索。名称支持模糊查询，域名仅支持等值查询，不支持模糊查询。
