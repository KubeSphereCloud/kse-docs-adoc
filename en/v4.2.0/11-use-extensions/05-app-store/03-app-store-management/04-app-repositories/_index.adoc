This section describes how to manage app repositories on the KubeSphere platform.

Add a Helm Chart repository as an app repository to the KubeSphere platform to install applications from the app repository in all projects. For information on how to create a Helm Chart repository, refer to link:https://helm.sh/zh/docs/topics/chart_repository/[Helm official documentation].
