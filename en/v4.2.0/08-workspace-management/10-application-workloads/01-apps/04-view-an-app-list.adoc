====
* 应用列表提供以下信息：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|应用的名称。

include::../../../../_custom/projectManagement/apps/apps-para-status_v4.adoc[]

|企业空间
|应用所属的企业空间。

|项目
|应用所属的项目。

|集群
|应用所属的集群。

|操作者
|创建应用的用户。

|应用类型
|应用模板的类型，如 **helm 应用**、**yaml 应用** 或 **operator 应用**。

|应用模板
|创建应用所使用的应用模板。仅对基于模板的应用显示。

|版本
|应用的版本号。

* 对于基于模板的应用，显示 Helm Chart 的版本。

// * 对于自制应用，显示用户自定义的版本号。

|更新时间
|应用的更新时间。
|===
--

* 在列表上方点击搜索框并输入关键字，可按名称搜索应用。
