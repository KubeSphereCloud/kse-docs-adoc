|The type of GPU.

|Scheduling Policy
|
* **Exclusive GPU**: A task exclusively occupies all computing and video memory resources of a physical GPU, thereby achieving optimal and stable performance.

* **Isolated Sharing**: A physical GPU is partitioned into multiple vGPUs through virtualization technology, allowing multiple tasks to share the GPU simultaneously while providing resource isolation between tasks.

|GPU Count
|The number of GPUs installed on the node.

|GPU Status
|The number of GPUs in normal and abnormal status.

|Creation Time
|The time when the node joined the current cluster.
