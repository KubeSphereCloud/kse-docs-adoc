+
====
* KubeSphere License
+
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Authorized Organization
|The organization authorized to use KubeSphere.

|Authorization Status
|
* **Unauthorized**: No KubeSphere license is obtained.
* **Authorization Expired**: The KubeSphere license has expired.
* **Insufficient Authorization Quantity**: The current cluster scale exceeds the authorized cluster scale for using KubeSphere.
* **Authorized**: KubeSphere authorization is normal.

// |Maintenance Status
// |
// * **No Maintenance**: No maintenance service is available.
// * **Expired**: The maintenance service has expired.
// * **Under Maintenance**: Within the maintenance service period.

|Cluster ID
|The current cluster ID, used to determine if the cluster is authorized.

|Authorization Type
|There are two types of authorization: Subscription Authorization and Perpetual Authorization.

**Subscription Authorization**: Community Edition-Subscription, Standard Edition-POC, Advanced Edition-POC, Enterprise Edition-POC, Standard Edition-Subscription, Advanced Edition-Subscription, Enterprise Edition-Subscription.

**Perpetual Authorization**: Standard Edition-Perpetual, Advanced Edition-Perpetual, Enterprise Edition-Perpetual.

|Authorized Scale
|The authorized cluster scale for using KubeSphere, measured in **vCPU**.

|Subscription Start Time
|The effective time of the product license. Displayed only when the Authorization Type is Subscription Authorization.

|Subscription End Time
|The expiration time of the product license. Displayed only when the Authorization Type is Subscription Authorization.

|Maintenance Expiration Time
|The expiration time of the maintenance service. Displayed only when the Authorization Type is Perpetual Authorization.
|===

* Extension License
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Extension Name
|The name of the extension.

|Authorization Status
|
* **Authorization Expired**: The license has expired.
* **Insufficient Authorization Quantity**: The current resource requirement exceeds the authorized resource limit.
* **Authorized**: Authorization is normal.

|Authorization Type
|There are two types of authorization: Subscription Authorization and Perpetual Authorization.

**Subscription Authorization**: Community Edition-Subscription, Standard Edition-POC, Advanced Edition-POC, Enterprise Edition-POC, Standard Edition-Subscription, Advanced Edition-Subscription, Enterprise Edition-Subscription.

**Perpetual Authorization**: Standard Edition-Perpetual, Advanced Edition-Perpetual, Enterprise Edition-Perpetual.

|Subscription Start Time
|The effective time of the product license. Displayed only when the Authorization Type is Subscription Authorization.

|Subscription End Time
|The expiration time of the product license. Displayed only when the Authorization Type is Subscription Authorization.

|Maintenance Expiration Time
|The expiration time of the maintenance service. Displayed only when the Authorization Type is Perpetual Authorization.
|===
