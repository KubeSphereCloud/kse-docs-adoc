[%header,cols="1a,4a"]
|===
|Parameter |Description

|Status
|The current status of the application version.

* **Not Submitted**: The application version has been uploaded to the workspace but has not been submitted to the KubeSphere platform's App Store administrator for review.

* **Submitted**: The application version has been submitted to the KubeSphere platform's App Store administrator for review.

* **Published**: The application version has been approved by the KubeSphere platform's App Store administrator and can be listed in the App Store.

* **Rejected**: The application version was not approved by the KubeSphere platform's App Store administrator and failed the review.

* **Unpublished**: The application version was unpublished by the App Store administrator after being published, or the application version was delisted from the KubeSphere platform's App Store by the App Store administrator.
