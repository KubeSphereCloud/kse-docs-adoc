本节介绍如何在集群和项目中创建、编辑、启用、禁用和查看告警规则组。

创建规则组后，系统可在特定监控指标满足预设条件和持续时间时生成告警。

//note
[.admon.note,cols="a"]
|===
|说明

|
- KubeSphere 为集群提供了内置规则组，同时也支持自定义规则组。若已安装 WizTelemetry 全局监控扩展组件，内置规则组只能在 WizTelemetry 可观测平台的全局告警中进行管理。有关更多信息，请参阅link:../../07-whizard/05-alert-management/01-rule-groups/[全局告警]。

- 在项目中，只支持自定义规则组。
|===
