:ks_navigation: **全局监控 > 企业空间**


本节介绍如何查看单个企业空间的监控详情。


== 前提条件

您需要在 KubeSphere 平台具有 **platform-admin** 角色。有关更多信息，请参阅link:../../../../../05-users-and-roles/01-users/[用户]和link:../../../../../05-users-and-roles/02-platform-roles/[平台角色]。


== 操作步骤

include::../../../../../_custom/platformManagement/extensionManagement/extensionManagement-oper-logInAsPlatformAdmin.adoc[]

+

include::../../../../../_custom/WizTelemetry/enter-into-WizTelemetry.adoc[]

+

include::../../../../../../_ks_components/oper-navigate.adoc[]

. 在企业空间列表中点击一个企业空间的名称打开其详情页面。

. 在详情页面的上方区域，点击**管理企业空间**可进入当前企业空间的管理页面。

. 在详情页面的**概览**页签下，查看企业空间的概览信息。
+
--
[%header,cols="1a,4a"]
|===
|功能区 |描述

|资源统计
|当前企业空间的授权集群数量，已创建的项目、部署、有状态副本集、守护进程集、任务、服务、持久卷声明、应用路由、容器组的数量。

|实时资源用量
|当前企业空间 CPU、内存的实时用量和总量；以及持久卷用量和持久卷预留数量。点击**全部**，可查看不同类型的存储的持久卷用量和持久卷预留数量。

点击相应区域，可查看该资源的实时用量百分比。

|企业空间配额
|
ifeval::["{file_output_type}" == "html"]
有关参数的更多信息，请参阅link:../../../../../08-workspace-management/06-workspace-settings/02-workspace-quotas/02-view-workspace-quotas/[查看企业空间配额]。
endif::[]

ifeval::["{file_output_type}" == "pdf"]
有关参数的更多信息，请参阅《KubeSphere 企业空间和项目管理指南》的“企业空间配额”章节。
endif::[]

|容器组
|当前企业空间中各种类型容器组的数量。

|===
--
