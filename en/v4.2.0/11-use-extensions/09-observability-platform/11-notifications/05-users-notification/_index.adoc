:ks_permission: **Receive Notifications**

This section describes how to set up the notification channel for the current user to receive alert messages from their project.

[.admon.note,cols="a"]
|===
|Note

|Users must have the pass:a,q[{ks_permission}] permission in their project to receive alert messages from that project.

|===
