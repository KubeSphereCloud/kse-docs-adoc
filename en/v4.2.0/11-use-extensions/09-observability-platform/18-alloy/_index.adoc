Grafana Alloy 是一个灵活、高性能、厂商中立的 OpenTelemetry Collector 分发版。Alloy 独特地结合了社区中最优秀的开源可观测性信号。Alloy 为 OTel、Prometheus、Pyroscope、Loki 以及许多其他指标、日志、追踪和性能分析工具提供原生管道。此外，您可以使用 Alloy 管道执行不同的任务，例如在 Loki 和 Mimir 中配置告警规则。Alloy 与 OTel Collector、Prometheus Agent 和 Promtail 完全兼容。您可以将 Alloy 作为这些解决方案的替代方案，或将其组合成包含多个采集器和代理的混合系统。您可以在 IT 基础设施的任何位置部署 Alloy，并将其与 Grafana LGTM 技术栈、Grafana Cloud 的遥测后端或任何其他供应商的兼容后端配对使用。Alloy 具有灵活性，您可以轻松配置它以适应本地部署、云端部署或两者混合部署的需求。更多信息，请参阅 link:https://grafana.com/docs/alloy/latest/[Alloy 官方文档]。

Grafana Alloy for WizTelemetry 可用于收集 Kubernetes 日志、Prometheus 指标、OpenTelemetry 数据，还可以用作导出各种指标的 Exporter。
