This section introduces how to manage network policies.

Network policies are used to control the inbound and outbound access permissions of pods within a cluster, allowing for network isolation within the same cluster. You can use network policies to achieve the following purposes:

* Allow pods to access only specific other pods or network segments.

* Allow pods to be accessed only by specific other pods or network segments.
