本节介绍如何编辑项目角色信息。

您可以编辑由用户创建的项目角色的别名和描述。KubeSphere 不支持编辑预置的项目角色，同时不支持编辑项目角色的名称。

. 在**编辑信息**对话框，设置项目角色的别名和描述，然后点击**确定**。
