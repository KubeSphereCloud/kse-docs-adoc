[%header,cols="1,5"]
|===
|Parameter |Description

|Username
a|Custom user name.

Configuration requirements: Username can only contain lowercase letters, numbers, and underscores (_), must start with a lowercase letter, and be between 2 and 26 characters in length.

|Password
a|Enter the user password.

Configuration requirements: Can only contain uppercase and lowercase letters (A～Z, a～z), numbers (0～9), special characters (!@#$%^&*_+-=), length must be between 8 and 32 characters.

|User Permission
|Select the account permission for the user. Supports Normal Permission or Advanced Permission.

|===
