:ks_navigation: **全局监控 >  项目**


本节介绍如何查看单个项目的监控详情。


== 前提条件

您需要在 KubeSphere 平台具有 **platform-admin** 角色。有关更多信息，请参阅link:../../../../../05-users-and-roles/01-users/[用户]和link:../../../../../05-users-and-roles/02-platform-roles/[平台角色]。


== 操作步骤

+

include::../../../../../_custom/WizTelemetry/enter-into-WizTelemetry.adoc[]

+

include::../../../../../../_ks_components/oper-navigate.adoc[]

. 在项目列表中点击一个项目的名称打开其详情页面。

. 在详情页面的上方区域，查看当前项目的基本信息。
+
--
* 点击**管理项目**可进入当前项目的详情页面。
* 点击image:/images/ks-qkcp/zh/icons/chevron-down.svg[chevron-down,18,18]可隐藏该区域。
--

. 在详情页面的**概览**页签下，查看项目的概览信息。
+
--
[%header,cols="1a,4a"]
|===
|功能区 |描述

|资源统计
|当前项目中已创建的部署、有状态副本集、守护进程集、任务、服务、持久卷声明、应用路由、容器组的数量。

|实时资源用量
|当前项目 CPU、内存的实时用量和总量；以及持久卷用量和持久卷预留数量。点击**全部**，可查看不同类型的存储的持久卷用量和持久卷预留数量。

点击相应区域，可查看该资源的实时用量百分比。

|项目配额
|
ifeval::["{file_output_type}" == "html"]
有关参数的更多信息，请参阅link:../../../../../08-workspace-management/08-projects/05-project-quotas/03-view-project-quotas/[查看项目配额]。
endif::[]

ifeval::["{file_output_type}" == "pdf"]
有关参数的更多信息，请参阅《KubeSphere 企业空间和项目管理指南》的“项目配额”章节。
endif::[]

|项目告警
|当前项目中全局告警规则组产生的告警数量。此处显示的告警不包括集群和项目的告警规则组产生的告警。全局告警规则组由平台管理员在**全局告警**中管理。

告警级别的类型包括**提醒、警告、重要和紧急**。

告警状态的类型包括：

include::../../../../../_custom/clusterManagement/alerts/alerts-para-alertingStatus.adoc[]

|容器组
|当前项目的各种类型容器组的数量。

容器组状态类型包括：

include::../../../../../_custom/clusterManagement/pods/pods-para-podStatus_overview.adoc[]

容器组 QoS（服务质量）类型包括：

include::../../../../../_custom/clusterManagement/pods/pods-para-qosTypes.adoc[]

include::../../../../../_custom/clusterManagement/pods/pods-para-oomTypes.adoc[]
|===
--

. 点击详情页面的**工作负载**、**容器组**页签，查看项目中所有工作负载、容器组的监控信息。
+
--
include::../../../../../_custom/WizTelemetry/sort-by-fields-and-order.adoc[]

* 在列表上方点击搜索框并输入关键字，可按名称搜索相应对象。

include::../../../../../../_ks_components/oper-customizeColumns.adoc[]
--

. 点击详情页面的**监控**页签，查看监控指标在指定时间范围内的详细信息。
