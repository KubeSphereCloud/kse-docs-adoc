RadonDB DMP is a newly released enterprise-grade cloud-native database management platform from QingCloud. Built upon the KubeSphere LuBan cloud-native extensible open architecture, RadonDB DMP can easily achieve cross-cloud deployment, multi-cloud management, intelligent observability, and unified operations for multiple types of mainstream databases. It fully constructs an ecosystem closed-loop for scenarios such as high performance, high availability, business tracking, and data flow, helping customers truly achieve Database as a Service.

After installing the RadonDB DMP extension, click the image:/images/ks-qkcp/zh/icons/grid.svg[grid,18,18] icon in the top navigation bar, then click RadonDB DMP to enter the database management platform.

For details on how to use the RadonDB DMP database management platform, please refer to the link:https://radondb-docs.kubesphere.com.cn/[RadonDB DMP User Guide].
