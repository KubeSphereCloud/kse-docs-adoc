本节介绍如何编辑镜像构建器信息。

您可以编辑镜像构建器的别名和描述。KubeSphere 不支持编辑镜像构建器的名称。

. 在需要操作的镜像构建器右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**编辑信息**。

. 在**编辑信息**对话框，设置镜像构建器的别名和描述，然后点击**确定**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|别名
|镜像构建器的别名。不同镜像构建器的别名可以相同。
// 别名只能包含中文、字母、数字和连字符（-），不得以连字符（-）开头或结尾，最长 63 个字符。

|描述
|镜像构建器的描述信息。描述可包含任意字符，最多包含 256 个字符。
|===
