. 查看 KubeSphere 和扩展组件的许可证。
+
====
* KubeSphere 许可
+
[%header,cols="1a,4a"]
|===
|参数 |描述

|授权组织
|被授权使用 KubeSphere 的组织。

|授权状态
|
* **未授权**：未获得 KubeSphere 的许可证。
* **授权已过期**： KubeSphere 的许可证已过期。
* **授权数量不足**：当前集群规模超出被许可使用 KubeSphere 的集群规模。
* **已授权**： KubeSphere 授权正常。

// |维保状态
// |
// * **无维保**：不享受维保服务。
// * **已过期**：维保服务已过期。
// * **维保中**：在维保服务期内。

|集群 ID
|当前集群 ID，用于判断集群是否享有授权。

|授权类型
|授权分为两类：订阅授权和永久授权。

**订阅授权**：社区版-订阅、标准版-POC、高级版-POC、旗舰版-POC、标准版-订阅、高级版-订阅、旗舰版-订阅。

**永久授权**：标准版-永久、高级版-永久、旗舰版-永久。

|授权规模
|被许可使用 KubeSphere 的集群规模，单位为 **vCPU**。

|订阅生效时间
|产品许可证的生效时间。仅在授权类型为订阅授权时展示。

|订阅截止时间
|产品许可证的过期时间。仅在授权类型为订阅授权时展示。

|维保到期时间
|维保服务的过期时间。仅在授权类型为永久授权时展示。
|===

* 扩展组件许可
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|组件名称
|扩展组件的名称。

|授权状态
|
* **授权已过期**：许可证已过期。
* **授权数量不足**：当前的资源需求超出被许可使用的资源上限。
* **已授权**：授权正常。

|授权类型
|授权分为两类：订阅授权和永久授权。

**订阅授权**：社区版-订阅、标准版-POC、高级版-POC、旗舰版-POC、标准版-订阅、高级版-订阅、旗舰版-订阅。

**永久授权**：标准版-永久、高级版-永久、旗舰版-永久。

|订阅生效时间
|产品许可证的生效时间。仅在授权类型为订阅授权时展示。

|订阅截止时间
|产品许可证的过期时间。仅在授权类型为订阅授权时展示。

|维保到期时间
|维保服务的过期时间。仅在授权类型为永久授权时展示。
|===

* 在列表上方点击搜索框并设置搜索条件，可按组件名称搜索组件的许可证信息。
