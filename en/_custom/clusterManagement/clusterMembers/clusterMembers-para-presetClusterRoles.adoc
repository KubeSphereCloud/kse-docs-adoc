The KubeSphere platform provides the following built-in cluster roles:

[%header,cols="1a,4a"]
|===
|Role |Description

|cluster-admin
|Cluster administrator, has all permissions within the cluster except for creating and deleting the cluster.

|cluster-viewer
|Cluster viewer, has view permissions for all resources within the cluster.
|===
