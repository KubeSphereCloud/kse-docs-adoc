本节介绍如何登录 KubeSphere Web 控制台并进入联邦项目。


== 前提条件

* KubeSphere 平台需要安装并启用**联邦集群应用管理**扩展组件。

* 您需要获取联邦项目成员用户名和密码。


== 方法 1

. 登录 KubeSphere Web 控制台并进入您的企业空间。

. 在左侧导航栏点击**联邦项目**进入联邦项目页面。

. 在联邦项目页面，点击一个联邦项目的名称进入联邦项目。

== 方法 2

. 登录 KubeSphere Web 控制台。

. 点击右上角的**组件坞 > 联邦项目**进入联邦项目页面。

. 在联邦项目页面，选择一个企业空间，查看对应的联邦项目。

. 点击一个联邦项目的名称进入联邦项目。
