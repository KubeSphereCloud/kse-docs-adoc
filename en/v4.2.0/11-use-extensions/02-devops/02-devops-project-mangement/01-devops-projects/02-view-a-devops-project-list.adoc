* The DevOps project list provides the following information:
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name of the DevOps project.

|Status
|The current status of the DevOps project.

* **Successful**: The DevOps project has been created successfully and is available.

* **Pending**: The DevOps project is being created.

* **Deleting**: The DevOps project is being deleted.

|Creator
|The user who created the DevOps project.

|Creation Time
|The creation time of the DevOps project.
|===
--

* Click the search box above the list and enter a keyword to search for DevOps projects by name and alias.
