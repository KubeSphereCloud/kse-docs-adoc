|Status
|The current status of the application template.

* **Unpublished**: The application template has been created successfully but has not been published to the KubeSphere platform's App Store.

* **Published**: The application template has been created successfully and has been published to the KubeSphere platform's App Store.

* **Suspended**: The application template has been suspended by the App Store administrator after being published to the KubeSphere platform's App Store.
