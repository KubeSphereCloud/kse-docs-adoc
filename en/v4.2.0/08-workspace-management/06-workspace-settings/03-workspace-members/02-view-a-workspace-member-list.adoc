* The workspace member list provides the following information:
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The username of the workspace member.

|Status
|
The current status of the workspace member.

include::../../../../_custom/clusterManagement/clusterMembers/clusterMembers-para-userStatus.adoc[]

|Role
|The role of the workspace member within the workspace.

|Last Login
|The time when the workspace member last logged in to the KubeSphere web console.
|===
--

* Click the search box above the list and enter a keyword to search for workspace members by username.
