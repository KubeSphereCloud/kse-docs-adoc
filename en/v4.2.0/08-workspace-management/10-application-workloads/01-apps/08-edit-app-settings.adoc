This section describes how to edit the settings of a created app.

You can also edit app settings when creating an app. For more information, see link:../01-install-an-app-from-the-app-store[Install an App from the App Store] and link:../02-install-an-app-using-an-app-template[Install an App Using an App Template].

. In the **Edit App Settings** dialog box, select the target app version on the right, edit the app settings, and then click **OK**.
