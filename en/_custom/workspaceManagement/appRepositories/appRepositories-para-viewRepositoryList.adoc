* 应用仓库列表提供以下信息：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|应用仓库的名称。

|状态
|应用仓库当前的状态。

* **成功**：应用仓库已添加并同步成功。

* **同步中**：应用仓库正在与 Helm Chart 仓库同步中。

* **不同步**：同步间隔时间设置为 0 时，应用仓库不与 Helm Chart 仓库同步。

* **失败**：应用仓库与 Helm Chart 仓库同步失败。

|URL
|Helm Chart 仓库的 URL。

|定时同步
|是否与 Helm Chart 仓库定时同步，定时同步的时间间隔。

|类型
|系统仓库或自有仓库。

* 系统仓库：“应用商店管理”扩展组件中添加的应用仓库。
* 自有仓库：企业空间中添加的应用仓库。
|===
