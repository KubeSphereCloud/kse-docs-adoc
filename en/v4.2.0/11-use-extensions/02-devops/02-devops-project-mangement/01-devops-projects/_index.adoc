This section introduces how to manage DevOps projects.

DevOps projects provide users with continuous integration and continuous deployment (CI/CD) functionality. You can connect KubeSphere with third-party code repositories within a DevOps project, and then use pipelines or continuous deployment to automatically update source code changes to the target environment.
