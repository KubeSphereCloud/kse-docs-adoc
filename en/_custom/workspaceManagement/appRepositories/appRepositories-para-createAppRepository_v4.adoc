[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name of the application repository.

|Alias
|The alias of the application repository.

|URL
|The URL of the Helm Chart repository. Click **Verify** to test the availability of the Helm Chart repository.

|Sync Interval
|The automatic synchronization interval between the application repository and the Helm Chart repository. The value ranges from 3 minutes to 24 hours. The default value **0** means no automatic synchronization.

|Description
|The description of the application repository. The description can contain any characters, with a maximum length of 256 characters.
|===
