====
* 应用负载模板列表提供以下信息：
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|应用负载模板的名称和描述信息。

|类型
|应用负载模板的类型。

|项目
|应用负载模板所属的项目。

|创建时间
|应用负载模板的创建时间。
|===
--

* 在列表左上角的下拉列表中选择一个项目，可以查看特定项目中的应用负载模板。

* 在列表上方点击搜索框并设置搜索条件，可按名称和类型搜索应用负载模板。
