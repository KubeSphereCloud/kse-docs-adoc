. Log in to the KubeSphere web console.

. In the upper-right corner of the page, click the current username, and then select **User Settings** from the drop-down list.

. On the **Basic Information** tab, set the user's email and language.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Email
|The user's email address. The email address can be used to log in to the KubeSphere web console.

|Language
|The language in which the KubeSphere web console is displayed for the current user.
|===
--

. Click the **Password Settings** tab to modify the current user's password.
+
--
The password must contain numbers, uppercase letters, and lowercase letters, and be 8 to 64 characters in length.
--
