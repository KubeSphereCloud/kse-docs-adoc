本节介绍如何使用“WizTelemetry 全局监控”扩展组件。

WizTelemetry 全局监控是 WizTelemetry 可观测平台中提供跨集群资源监控、多集群告警的扩展组件，包含原有 Whizard 可观测中心的功能。您可以在 WizTelemetry 可观测平台查看所有集群的汇总监控数据和每个集群的具体监控数据，同时可以在平台层级创建全局告警规则组并查看系统生成的全局告警。

安装“WizTelemetry 全局监控”扩展组件后，组件坞中的 WizTelemetry 可观测平台将显示**全局监控**和**全局告警**菜单。

[.admon.attention,cols="a"]

|
