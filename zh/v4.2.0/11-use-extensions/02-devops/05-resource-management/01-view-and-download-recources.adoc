本节介绍如何通过目录树的形式查看项目中的 Kubernetes 资源。


== 前提条件

* KubeSphere 平台需要安装并启用 **DevOps** 扩展组件。

* 已创建一个企业空间、一个 DevOps 项目和一个用户（例如 **project-regular**），并已邀请该用户至 DevOps 项目且授予 **operator** 角色。请参阅link:../../02-devops-project-mangement/03-project-members/01-invite-a-user-to-a-project/[邀请用户加入 DevOps 项目]。


== 操作步骤

. 点击搜索资源文件，或点击目录名称，在右侧窗口查看目录中的文件。
+
--
* 点击目录左侧的image:/images/ks-qkcp/zh/icons/triangle-right.svg[,18,18]可展开单个目录。

* 点击**全部资源**右侧的image:/images/ks-qkcp/zh/icons/fold.png[,18,18]可收起所有目录，点击image:/images/ks-qkcp/zh/icons/refresh-new.png[,18,18]刷新目录。
--

. 在右侧窗口点击目录中的文件，查看文件。
+
--
* 点击上方的image:/images/ks-qkcp/zh/icons/collapse1.svg[,18,18]可收起目录树区域。再次点击image:/images/ks-qkcp/zh/icons/expand1.svg[,18,18]可展开目录树区域。

* 点击右上角的image:/images/ks-qkcp/zh/icons/download-light.png[,18,18]可下载当前资源文件。
--
