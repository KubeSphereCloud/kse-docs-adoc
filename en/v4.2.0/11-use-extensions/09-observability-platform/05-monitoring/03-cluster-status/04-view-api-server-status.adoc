. On the **Cluster Monitoring** page, click the **API Server Monitoring** tab to view the operational status of the API server.
+
====
* The **API Server Monitoring** tab provides the following information:
+
--
[%header,cols="1a,4a"]
|===
|Metric |Description

|Request Latency
|The request latency of the API server within the specified time range.

|Requests per Second
|The number of requests per second accepted by the API server within the specified time range.
|===
