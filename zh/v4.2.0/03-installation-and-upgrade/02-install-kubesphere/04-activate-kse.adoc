登录{ks_product_left} Web 控制台后，您需要导入许可证（license），激活{ks_product_both}和扩展组件，以体验所有功能。

[.admon.note,cols="a"]
|===
|说明

|
若您是离线安装的{ks_product_left}，请联系交付服务专家直接获取 license。
|===

== 步骤 1：购买并获取 license

若您是在线安装的{ks_product_left}，请访问 link:https://kubesphere.com.cn/[{ks_product_right}官网]，点击**联系我们**，通过微信或 slack 咨询{ks_product_both}和扩展组件 license 的购买方式和获取方式。

image:/images/ks-qkcp/zh/v4.1.3/contact-us.png[, 70%]

== 步骤 2：导入 license

ifeval::["{file_output_type}" == "html"]
待获得 license 后，请参阅link:../../../04-platform-management/05-platform-settings/02-license/03-add-a-license/[添加许可证]导入{ks_product_both}和扩展组件的 license。
endif::[]

ifeval::["{file_output_type}" == "pdf"]
待获得 license 后，请参阅《{ks_product_right}平台管理指南》的“许可证”章节导入{ks_product_both}和扩展组件的 license。
endif::[]


== 步骤 3：验证激活

若许可证页面显示{ks_product_both}和扩展组件**已授权**，即表示激活成功，可开始使用{ks_product_left}。
