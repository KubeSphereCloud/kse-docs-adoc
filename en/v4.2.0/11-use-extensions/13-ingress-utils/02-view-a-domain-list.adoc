This section describes how to view the list of domains for which "Domain Reuse Validation" is enabled.


== Prerequisites

* The **KubeSphere Application Routing Tool** extension must be installed and enabled on the KubeSphere platform.

* The domain list provides the following information:
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name and description of the domain.

|Domain
|The domain address, supporting wildcard matching.
|===
--

* Click the search box above the list and set search criteria to search by name and domain. Name supports fuzzy search, while domain only supports exact match search and does not support fuzzy search.
