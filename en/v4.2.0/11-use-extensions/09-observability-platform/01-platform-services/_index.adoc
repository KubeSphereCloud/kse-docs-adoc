WizTelemetry 平台服务是 WizTelemetry 可观测平台中各扩展组件的公共服务。它是各个可观测性扩展组件通用的 APIServer，为所有可观测性扩展组件提供公共的后端平台服务。

// 安装 WizTelemetry 平台服务后，**组件坞**中将显示 **WizTelemetry 可观测平台**。

**配置说明：**

* 修改 WizTelemetry 平台服务的扩展组件配置，可配合 WizTelemetry 全局监控扩展组件，启用全局监控和全局告警功能。有关更多信息，请参阅 link:../07-whizard/01-enable-whizard/[WizTelemetry 全局监控]。

* 为 WizTelemetry 日志、WizTelemetry 审计、WizTelemetry 事件、以及通知历史配置 OpenSearch 日志接收器时，如果需要使用多个 OpenSearch 数据库，请参阅扩展中心“WizTelemetry 平台服务”扩展组件的详情页说明。
