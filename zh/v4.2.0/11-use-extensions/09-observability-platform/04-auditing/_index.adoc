本节介绍如何使用“WizTelemetry 审计”扩展组件。

该组件实时记录在 KubeSphere 平台上的用户相关操作行为，并提供多租户视角的审计历史的检索及查看功能，可快速回溯相关用户的操作行为。

安装“WizTelemetry 审计”扩展组件后，

* 组件坞中的 WizTelemetry 可观测平台将显示**日志**菜单，**日志**菜单下将显示**审计查询**选项。
* 集群左侧导航栏将显示**日志**菜单，**日志**菜单下将显示**日志接收器**和**审计查询**选项。**日志接收器**页面将显示**审计日志**页签，支持添加多种类型的日志接收器。有关更多信息，请参阅link:../02-logging/02-log-receivers/[日志接收器]。**审计查询**页面支持用户查询自身权限范围内的审计日志。

|===
| 注意

|
WizTelemetry 审计依赖 WizTelemetry 平台服务和 WizTelemetry 数据流水线，因此在安装 WizTelemetry 审计之前必须先安装并配置好 WizTelemetry 平台服务和 WizTelemetry 数据流水线两个扩展组件，否则审计功能无法正常使用！
