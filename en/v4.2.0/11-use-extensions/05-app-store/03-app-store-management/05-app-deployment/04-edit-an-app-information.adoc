This section describes how to edit the information of a deployed application.

You can edit the alias and description of an application. KubeSphere does not support editing the name of a deployed application.

== Prerequisites

* {empty}
include::../../../../../_custom/platformManagement/platformManagement-prer-requiredPermission_v4.adoc[]

:relfileprefix: ./

* The **KubeSphere App Store Management** extension must be installed and enabled on the KubeSphere platform.

. In the **Edit Information** dialog box, set the alias and description for the application, then click **OK**.
