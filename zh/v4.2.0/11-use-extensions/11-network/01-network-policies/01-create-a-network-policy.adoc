本节介绍如何创建网络策略。

[.admon.note,cols="a"]
|===
|说明

|在集群页面创建网络策略后，您可以进入网络策略对应项目所属的企业空间，点击**服务网络 > 项目网络策略**，选择项目，然后查看该项目的网络策略。

|===

[.admon.attention,cols="a"]
|===
|注意

|项目网络策略页面仅支持查看网络策略。如需创建或管理网络策略，请前往集群的“网络策略”页面操作。
|===
