This section introduces how to manage Persistent Volume Claims and Persistent Volumes.

A Persistent Volume Claim defines the requirements for storage resources. The system will create storage areas in the backend storage system based on the Persistent Volume Claim for use by containers. Each storage area corresponds to a Persistent Volume on the KubeSphere platform.
