资源管理提供租户视角的物理集群下的 Kubernetes 资源管理、通过目录树的形式预览各类资源，支持查看、下载资源的 YAML 文件，并同步到代码仓库。

// [.admon.note,cols="a"]
// |===
// |说明

// |此功能仅在 KubeSphere ``标准版、高级版和旗舰版``中提供。

// |===
