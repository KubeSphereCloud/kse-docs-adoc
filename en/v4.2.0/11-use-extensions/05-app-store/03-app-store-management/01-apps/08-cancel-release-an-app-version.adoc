本节介绍如何取消发布应用版本。

应用模板上架后，若取消发布其中的应用版本，该版本将被从应用商店中移除。若应用模板中的应用版本均为**取消发布**状态，该应用模板将自动下架。

应用版本取消发布后，应用商店管理员和企业空间成员可以删除该应用版本。
