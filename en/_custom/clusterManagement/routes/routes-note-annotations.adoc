KubeSphere's cluster gateway and project gateway are implemented based on the Nginx Ingress Controller. You can set annotations on the Ingress to control the gateway's behavior. For more information, please refer to the link:https://kubernetes.github.io/ingress-nginx/user-guide/nginx-configuration/annotations/[Nginx Ingress Controller official documentation].
