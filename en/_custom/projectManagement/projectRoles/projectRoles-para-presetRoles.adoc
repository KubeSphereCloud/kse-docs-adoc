KubeSphere provides the following preset project roles:

[%header,cols="1a,4a"]
|===
|Role |Description

|admin
|Project administrator, has all permissions within the project.

|operator
|Project operator, has all permissions within the project except for project settings management, role management, and member management.

|viewer
|Project viewer, has resource viewing permissions within the project.
|===
