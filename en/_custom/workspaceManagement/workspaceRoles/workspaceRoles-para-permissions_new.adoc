All workspace members can view basic workspace information. The permissions required for other operations are as follows:

* Projects
+
--
[%header,cols="1a,4a"]
|===
|Permission |Allowed Operations

|Project Creation
|Create projects and become the administrator of the created projects.

|Project Management
|Create, edit, and delete projects within the workspace.

|Project Viewing
|View the project list and view all resources within a project.
|===
--

* DevOps Projects (Requires installation of the DevOps extension)
+
--
[%header,cols="1a,4a"]
|===
|Permission |Allowed Operations

|DevOps Project Creation
|Create DevOps projects and become the administrator of the created DevOps projects.

|DevOps Project Management
|Create, edit, and delete DevOps projects within the workspace.

|DevOps Project Viewing
|View the DevOps project list and view all resources within a DevOps project.
|===
--

// * Applications
// +
// --
// [%header,cols="1a,4a"]
// |===
// |Permission |Allowed Operations

// |Application Repository Management
// |Add application repositories, edit application repositories, synchronize application repositories and Helm Chart repositories, delete application repositories.

// |Application Repository Viewing
// |View the application repository list.

// |Application Template Management
// |Create application templates, edit application template information, install applications using application templates, upload application versions, replace application versions, delete application versions, publish application versions to the App Store, delete application templates.

// |Application Template Viewing
// |View the application template list, view application template details, download Helm Chart archives.
// |===
// --

* Access Control
+
--
[%header,cols="1a,4a"]
|===
|Permission |Allowed Operations

// |Department Management
// |Set up department organizational structure, assign users to departments, remove department members.

// |Department Viewing
// |View the department organizational structure.

// |Member Management
// |Invite users to join the workspace, modify the roles of workspace members, remove workspace members.

|Member Viewing
|View the workspace member list, view workspace member details.

// |Role Management
// |Create workspace roles, edit workspace role information, edit workspace role permissions, delete workspace roles.

|Role Viewing
|View the workspace role list, view workspace role details.
|===
--

// * Workspace Settings
// +
// --
// [%header,cols="1a,4a"]
// |===
// |Permission |Allowed Operations

// |Workspace Settings Management
// |Edit workspace information, enable and disable network isolation, delete the workspace, edit workspace quotas.

// |Workspace Settings Viewing
// |View cluster resource usage, view project resource usage rankings, view workspace resource usage, view workspace quotas.
// |===
