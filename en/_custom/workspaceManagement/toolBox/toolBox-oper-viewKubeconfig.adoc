. Log in to the KubeSphere web console.

. Hover over the image:/images/ks-qkcp/zh/icons/hammer.svg[hammer,18,18] icon in the lower-right corner of the page, then select **kubeconfig** from the menu.

. View the kubeconfig configuration information in the pop-up dialog.
+
--
* The access address displayed in the **clusters:cluster:server** field is a virtual IP address, which can only be accessed from within the cluster. To access it from outside the cluster, modify this address to the externally exposed address of kube-apiserver.

* The **contexts:cluster:namespace** field defaults to the project name **default**. Please change the value of this field to the name of the project that the current user actually has access to.

* Click image:/images/ks-qkcp/zh/icons/download-dark.svg[download-dark,18,18] to download the **kubeconfig.yaml** file to your local machine.
