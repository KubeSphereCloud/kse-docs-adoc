EdgeWize Edge Computing Platform is a cloud-edge collaborative infrastructure platform designed for edge computing scenarios. It is built on the popular Kubernetes container orchestration engine and integrates and optimizes key functional modules for edge computing scenarios through a pluggable architecture. It provides enterprise users with a practical, iterative, and scenario-oriented comprehensive edge computing solution, helping them quickly build an open, easy-to-use, and intelligent edge container application management platform to facilitate the edge transformation of their business.

After installing the EdgeWize extension, click the image:/images/ks-qkcp/zh/icons/grid.svg[grid,18,18] icon in the top navigation bar, then click **edgewize-io** to enter the EdgeWize Edge Computing Platform.

For instructions on how to use the EdgeWize Edge Computing Platform, please refer to the link:https://edgewize-docs.kubesphere.com.cn[EdgeWize User Guide].
