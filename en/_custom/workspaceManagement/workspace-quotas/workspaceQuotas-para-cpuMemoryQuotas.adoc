[%header,cols="1a,4a"]
|===
|Parameter |Description

|CPU Limit
|The maximum total CPU limit for all containers running in a single cluster within the workspace.

|CPU Reservation
|The maximum total CPU reservation for all containers running in a single cluster within the workspace.

|Memory Limit
|The maximum total memory limit for all containers running in a single cluster within the workspace.

|Memory Reservation
|The maximum total memory reservation for all containers running in a single cluster within the workspace.

|Persistent Volume Claim Limit
|The current capacity of Persistent Volume Claims (PVCs) already created in the workspace, and the maximum capacity of PVCs that can be created.

|Persistent Volume Claim
|The current number of Persistent Volume Claims (PVCs) already created in the workspace, and the maximum number of PVCs that can be created.
|===
