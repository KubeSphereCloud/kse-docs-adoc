:ks_permission: **Cluster Management**

This section describes how to add labels to clusters.

== Prerequisites

You need to have the pass:a,q[{ks_permission}] permission on the KubeSphere platform.

== Steps

include::../../../../_custom/platformManagement/platformManagement-oper-logIn.adoc[]

+

. Click **Cluster Management**.

+

include::../../../../_custom/platformManagement/multiclusterManagement/clusterTags/clusterTags-openListPage_v42.adoc[]

+

. To the right of the label you want to add to a cluster, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Add to Cluster** from the dropdown list.

. In the **Add Label to Cluster** dialog box, select the cluster(s) to which you want to add the label, then click **OK**.
