---
title: "升级后，针对原 Redis Cluster 实例的建议"
description: "介绍如何处理升级后 Redis Cluster 实例的管理问题。"
weight: 02
---

从 KubeSphere v4.1.2 升级到 v4.1.3 后，`redis-operator` 的版本发生了变化，不再管理由旧版本 redis-operator 创建的 Redis Cluster 实例。

在数据库管理平台，仍可以查看原有的 Redis Cluster 实例，但无法通过控制台执行扩容、增减节点等操作，新版本 redis-operator 也不会维护旧 Redis Cluster 实例的状态。

== 建议

. 备份旧 Redis Cluster 集群的数据。

. 重新创建 Redis Cluster 集群来替换旧集群。
