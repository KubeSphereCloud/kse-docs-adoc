* Application Workloads
+
--
[%header,cols="1a,4a"]
|===
|Permission |Allowed Operations

|Application Workloads View
|
* Application View: View application list, view template-based application details, view composed app details.

* Service View: View service topology, view service list, view service details.

* Workload View: View workload list, view workload details.

* Job View: View Job list, view Job details.

* Ingress View: View Ingress list, view Ingress details, access backend services of Ingress.

* Pod View: View pod list, view pod details, view container details.

* Grayscale Release View: View grayscale release task list, view grayscale release task details.

* Image Builder View: View image builder list, view image builder details.

|Application Workloads Management
|
* Application Management: Install applications from template, install applications from App Store, create composed apps, delete applications.

* Service Management: Create stateful or stateless services, create external services, create services from source code, create services from artifacts, create services from existing workloads, edit service information, edit service settings, edit service external access settings, edit monitoring exporters, delete services.

* Workload Management: Create workloads, edit workload information, recreate workloads, delete workloads, adjust pod replica count, stop workloads, roll back workload settings, edit autoscaling settings, edit workload settings.

* Job Management: Create Jobs, edit Job information, delete Jobs, rerun Jobs, suspend CronJobs.

* Ingress Management: Create Ingress, edit Ingress information, edit routing rules, edit Ingress annotations, delete Ingress.

* Pod Management: Access container terminal, delete pods.

* Grayscale Release Management: Create grayscale release tasks, switch active version (Blue-Green Deployment), adjust traffic distribution ratio (Canary Release), edit grayscale release task settings, delete grayscale release tasks.

* Image Builder Management: Create image builder from source code, create image builder from artifacts, edit image builder information, run image builder, delete image builder.
|===
