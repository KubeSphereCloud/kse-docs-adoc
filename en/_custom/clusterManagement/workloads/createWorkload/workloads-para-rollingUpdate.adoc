// tag::onlydeploy[]
设置滚动更新策略的相关参数。

部署：

[%header,cols="1a,4a"]
|===
|参数 |描述

|最大不可用容器组数量
|更新过程中允许的不可用容器组副本的最大数量或百分比。

|最大多余容器组数量
|更新过程中允许的多余容器组副本的最大数量或百分比。
|===
// end::onlydeploy[]

有状态副本集：

[%header,cols="1a,4a"]
|===
|参数 |描述

|容器组副本分组序号
|设置一个分组序号以将容器组副本分成两组。更新有状态副本集时，只有序号大于或等于分组序号的容器组副本会被更新。
|===

ifndef::multicluster[]
守护进程集：

[%header,cols="1a,4a"]
|===
|参数 |描述

|最大不可用容器组数量
|更新过程中允许的不可用容器组副本的最大数量或百分比。

|容器组就绪最短运行时长
|容器组副本被视为就绪所需要的最短稳定运行时长。
|===
endif::[]
