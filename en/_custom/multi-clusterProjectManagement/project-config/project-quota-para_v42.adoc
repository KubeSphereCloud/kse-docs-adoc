[%header,cols="1a,4a"]
|===
|参数 |描述

|CPU 上限
|项目中所有容器 CPU 上限总和的最大值，单位为核。

|CPU 预留
|项目中所有容器 CPU 预留总和的最大值，单位为核。

|内存上限
|项目中所有容器内存上限总和的最大值，单位为 MiB。

|内存预留
|项目中所有容器内存预留总和的最大值，单位为 MiB。

|GPU 上限
|项目中当前已使用的 GPU 数量，以及可以使用的 GPU 数量上限，单位为 GPU。

|持久卷声明限制
// |持久卷声明总容量限制
|企业空间中当前已创建的持久卷声明总容量，以及可以创建的持久卷声明的容量上限。

|持久卷声明
// |持久卷声明数量限制
|企业空间中当前已创建的持久卷声明总数量，以及可以创建的持久卷声明的数量上限。

|存储类关联资源限制
// |存储类关联持久卷声明总容量限制
|存储类允许创建的持久卷声明的容量上限。

|存储类关联资源
// |存储类关联持久卷声明数量限制
|存储类允许创建的持久卷声明的数量上限。

// |卷
// |项目中当前已创建的卷数量，以及可以创建的卷数量上限。

|容器组
|项目中当前已创建的容器组数量，以及可以创建的容器组数量上限。

|部署
|项目中当前已创建的部署数量，以及可以创建的部署数量上限。

|有状态副本集
|项目中当前已创建的有状态副本集数量，以及可以创建的有状态副本集数量上限。

|守护进程集
|项目中当前已创建的守护进程集数量，以及可以创建的守护进程集数量上限。

|任务
|项目中当前已创建的任务数量，以及可以创建的任务数量上限。

|定时任务
|项目中当前已创建的定时任务数量，以及可以创建的定时任务数量上限。

|服务
|项目中当前已创建的服务数量，以及可以创建的服务数量上限。

|应用路由
|项目中当前已创建的应用路由数量，以及可以创建的应用路由数量上限。

|保密字典
|项目中当前已创建的保密字典数量，以及可以创建的保密字典数量上限。

|配置字典
|项目中当前已创建的配置字典数量，以及可以创建的配置字典数量上限。
|===
