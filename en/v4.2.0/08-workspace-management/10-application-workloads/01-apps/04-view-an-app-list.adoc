====
* The application list provides the following information:
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Name
|The name of the application.

include::../../../../_custom/projectManagement/apps/apps-para-status_v4.adoc[]

|Workspace
|The workspace to which the application belongs.

|Project
|The project to which the application belongs.

|Cluster
|The cluster to which the application belongs.

|Creator
|The user who created the application.

|Application Type
|The type of the application template, such as **Helm Application**, **YAML Application**, or **Operator Application**.

|Application Template
|The application template used to create the application. Only displayed for template-based applications.

|Version
|The version number of the application.

* For template-based applications, the Helm Chart version is displayed.

// * For composed apps, the user-defined version number is displayed.

|Update Time
|The time when the application was updated.
|===
--

* Click the search box above the list and enter a keyword to search for applications by name.
