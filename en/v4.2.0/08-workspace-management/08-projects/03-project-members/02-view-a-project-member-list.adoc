[%header,cols="1a,4a"]
|===
|Parameter |Description

|Username
|The username of the project member.

|Status
|
The current status of the project member.

include::../../../../_custom/clusterManagement/clusterMembers/clusterMembers-para-userStatus.adoc[]

|The role of the project member within the project.

|Last Login
|The time when the project member last logged into the KubeSphere web console.
|===
