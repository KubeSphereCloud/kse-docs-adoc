. On the application template details page, click the **Application Review** tab to view the review records of the application versions.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Time
|The creation time of the application review record.

|Status
|The status of the application version corresponding to the application review record.

|Version
|The version of the Helm Chart and the application corresponding to the review record.

|Rejection Reason
|The reason provided by the App Store administrator for rejecting the application version after submission for review. This is only displayed when the application version is rejected by the App Store administrator.

|Operator
|The user who performed the operation on the application version.
|===
