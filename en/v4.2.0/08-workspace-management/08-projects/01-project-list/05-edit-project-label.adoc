This section describes how to edit project labels.

Project labels are identifiable key-value pairs added to projects, used for identification and grouping.

== Steps

include::../../../../_custom/workspaceManagement/workspaceManagement-oper-openWorkspacePage.adoc[]

+

include::../../../../../_ks_components/oper-navigate.adoc[]

. On the right side of the project you want to operate on, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Edit Labels** from the dropdown list.

. In the **Edit Labels** dialog box, set the label key-value pairs, then click **OK**.
+
--
* Click **Add** to set multiple labels.

* Click image:/images/ks-qkcp/zh/icons/trash-light.svg[trash-light,18,18] on the right side of an existing label to delete it.
