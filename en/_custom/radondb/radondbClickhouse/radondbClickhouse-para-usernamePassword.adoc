|===
|Parameter |Description

|Username
|The username can only contain uppercase letters, lowercase letters, numbers, and underscores (_). It must start with an uppercase or lowercase letter and be 2 to 26 characters in length.

|Password
|The password must contain at least one uppercase letter, one lowercase letter, one number, and one special character (!@#$%^&*_+-=). It must be 8 to 32 characters in length.
|===
