This section describes how to modify the current user's email information and the interface display language.

== Prerequisites

You have obtained the username and password for the KubeSphere web console.

== Steps

. Log in to the KubeSphere web console.

. In the upper-right corner of the page, click the current username, then select **User Settings** from the dropdown list.

. On the **Basic Information** tab, set the user's email and language.
+
--
[%header,cols="1a,4a"]
|===
|Parameter |Description

|Email
|The user's email address. The email address can be used to log in to the KubeSphere web console.

|Language
|The language in which the KubeSphere web console is displayed for the current user.
|===
