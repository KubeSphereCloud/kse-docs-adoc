本节介绍如何管理配置资源。

KubeSphere 支持管理以下配置资源：

* 保密字典：包含 Base64 编码的键值对，用于存储密码、令牌、密钥等保密数据。

* 配置字典：包含明文键值对，用于存储普通配置数据。

// * 服务帐户：存储当前 KubeSphere 集群的访问信息，用于向应用程序提供 KubeSphere 集群的访问权限。
