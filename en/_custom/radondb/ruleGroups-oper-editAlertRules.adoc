. In the **Edit Alert Rule** dialog box, you can manage alert rules.
+
--
* Add an alert rule: Click **Add Alert Rule** below an existing alert rule to create a new one.

* Enable/Disable an alert rule: Hover over an existing alert rule and click the toggle switch to enable/disable it.

* Delete an alert rule: Hover over an existing alert rule and click image:/images/ks-qkcp/zh/icons/trash-light.svg[trash-light,18,18] to delete it.

* Modify an alert rule: Hover over an existing alert rule and click image:/images/ks-qkcp/zh/icons/pen-light.svg[pen,18,18] to modify the settings of the alert rule.

--
. After completing the operations, click **OK**.
