本节介绍如何在项目中启用日志收集。
//note
[.admon.note,cols="a"]
|===
|说明

|
启用后，在项目中创建工作负载、服务和任务时，**存储设置**页签的**收集卷上日志**功能将可用。
|===


== 前提条件


:relfileprefix: ../../../../../
* {empty}
include::../../../../../_custom/projectManagement/projectManagement-prer-requiredPermission_v42.adoc[]

:relfileprefix: ./

* KubeSphere 平台需要安装并启用 **WizTelemetry 日志**扩展组件，并已link:../01-enable-logsidecar/[启用日志收集]。


== 操作步骤

include::../../../../../_custom/workspaceManagement/workspaceManagement-oper-openWorkspacePage.adoc[]

+

include::../../../../../../_ks_components/oper-navigate.adoc[]

+

. 在需要操作的项目右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**启用日志收集**。

. 在**启用日志收集**对话框，点击**确定**。
