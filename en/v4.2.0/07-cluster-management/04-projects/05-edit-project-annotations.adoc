This section describes how to edit project annotations.

Project annotations are key-value pairs that add additional metadata or descriptive information to a project.

== Steps

include::../../../_custom/clusterManagement/clusterManagement-oper-openClusterPage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]


. On the right side of the project you want to operate, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Edit Annotations** from the dropdown list.

. In the **Edit Annotations** dialog box, set the annotation key-value pairs, then click **OK**.
+
--
* Click **Add** to set multiple annotations.

* Click image:/images/ks-qkcp/zh/icons/trash-light.svg[trash-light,18,18] on the right side of a created annotation to delete it.
