. Transfer the KubeSphere installation package to any cluster node and log in to that node.

. Execute the following commands to extract the installation package and navigate to the directory generated after extraction (replace `<package name>` with the actual name of the installation package and `<directory>` with the directory generated after extraction):
+
--
// Bash
include::../../../_ks_components/code/bash.adoc[]

tar -zxvf <package name>

include::../../../_ks_components/code/codeEnd.adoc[]

// Bash
include::../../../_ks_components/code/bash.adoc[]

cd <directory>

include::../../../_ks_components/code/codeEnd.adoc[]
--

. Execute the following command to add execute permission to the KubeKey binary file **kk**:
+
--
// Bash
include::../../../_ks_components/code/bash.adoc[]

sudo chmod +x kk

include::../../../_ks_components/code/codeEnd.adoc[]
--
