Constraints are rules or conditions used to define and enforce cluster policies. They are used to declare Gatekeeper constraint templates that need to be enforced and are instances of constraint templates.

For more information about constraints, please refer to link:https://open-policy-agent.github.io/gatekeeper/website/docs/howto/#constraints[Constraints].
