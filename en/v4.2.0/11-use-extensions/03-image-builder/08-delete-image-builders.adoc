This section describes how to delete an image builder.

// Warning
include::../../../../_ks_components/admonitions/warning.adoc[]

Once deleted, the image builder cannot be recovered. Please perform this operation with caution.

include::../../../../_ks_components/admonitions/admonEnd.adoc[]

== Delete a Single Image Builder

include::../../../_custom/workspaceManagement/workspaceManagement-oper-openWorkspacePage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

include::../../../_custom/projectManagement/projects-oper-dropdownList.adoc[]

+

. On the right side of the image builder you want to delete, click image:/images/ks-qkcp/zh/icons/more.svg[more,18,18], then select **Delete** from the dropdown list.

. In the **Delete Image Builder** dialog box, enter the name of the image builder, then click **OK**.


== Batch Delete Image Builders

include::../../../_custom/workspaceManagement/workspaceManagement-oper-openWorkspacePage.adoc[]

+

include::../../../../_ks_components/oper-navigate.adoc[]

+

include::../../../_custom/projectManagement/projects-oper-dropdownList.adoc[]

+

. Select the checkbox on the left of the image builder(s) you want to delete, then select **Delete** above the image builder list.

. In the **Batch Delete Image Builders** dialog box, enter the name(s) of the image builder(s), then click **OK**.
