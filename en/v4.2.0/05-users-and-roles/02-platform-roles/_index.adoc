This section introduces how to manage platform roles.

Platform roles contain specific platform management or viewing permissions. You can assign roles containing specific permissions to users to control their permissions within the KubeSphere platform.

include::../../../../_ks_components/admonitions/note.adoc[]
The default administrator user **admin** has all permissions in the KubeSphere platform.
